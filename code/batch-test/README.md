# 测试集批量测试

保存了模型后，需要对模型测试，图片集路径为
```python
path = '../../data/test-set/*.png'
```  

导入模型后，测试的原理就是替换，把训练的图片改为测试的图片。  
模型中还报错了最后一个全连接层的输出logits，把其从模型中读取出来，更改图片，再运行模型几个得出结果。
  
  
测试集图片命名
```bash
路飞 1xx.jpg
罗宾 2xx.jpg
娜美 3xx.jpg
乔巴 4xx.jpg
索隆 5xx.jpg
```
> 就是打个标签而已，paths[i][-7:] 读取出1,2,3…，和模型结果对比即可。
